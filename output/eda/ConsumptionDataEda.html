Maybe if we had more specific consumption data (per day instead of per month) the correlation would be higher, but for now we can conclude the temperature does not have a significant effect on the energy consumption.

Conclusion¶

Right now this dataset is unfortunately not very useful for our project. The only energy consumption data we could find was aggregated per month and the production and weather data we have are both hourly. To make the consumption data more useful for our project and to compare it to the production data we would need hourly consumption data. This way we also cannot calculate or predict energy congestion. The source of the production data we used mentioned that energy consumption data will be available in the future, so the new data would have to be collected when it is available and the project could be completed once it is available. For this reason we will only focus on the energy production data during this challenge.
